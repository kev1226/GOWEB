# Proyecto "Hola Mundo" en Go

Este es un proyecto simple en Go que crea una página web con el mensaje "¡Hola Mundo!".

## Pasos para ejecutar

1. Asegúrate de tener Go instalado en tu máquina. Si no lo tienes, puedes descargarlo desde [https://go.dev/dl/](https://go.dev/dl/).
2. Clona este repositorio en tu máquina local.
3. Navega al directorio del proyecto en tu terminal.
4. Ejecuta el siguiente comando para iniciar el servidor:

   ```bash
   go run app.go
